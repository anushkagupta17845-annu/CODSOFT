# 🛍️ Product Recommendation System

A simple Product Recommendation System built using Python, Pandas, and Streamlit. The application recommends products based on category, budget, and minimum rating.

## 📌 Features

- Product category selection
- Budget-based filtering
- Rating-based filtering
- Product ranking using a recommendation score
- Interactive Streamlit interface
- CSV-based dataset

## 🛠️ Technologies Used

- Python
- Pandas
- Streamlit
- Git & GitHub

## 🚀 Installation

Clone the repository:

bash
git clone https://github.com/anushkagupta17845-annu/CODSOFT.git


Move into the project folder:

bash
cd CODSOFT


Install dependencies:

bash
pip install pandas streamlit


## ▶️ Run Terminal Version

bash
python app_windows.py


## ▶️ Run Streamlit Version

bash
streamlit run streamlit_app_windows.py
